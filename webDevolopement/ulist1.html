Unordered list

What is unordered list?

An unordered list is a list in which the order of data is not necessary. This list is especially used to list up the random data. For example likes of a person or list of food available in hotel etc. now lets look how to create this list using HTML.

Creating HTML unordered list:

To Create a list we have <ul> tag. And to add the elements in the lists there is <li> tag. In this <li> tag add elements. Add anather <li> to add anather element to the list and carry on the same process as you want.

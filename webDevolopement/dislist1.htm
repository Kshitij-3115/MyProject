Discription list

What is Discription list?

Discription list is a list in which a term and its discription is listed. This list is especially used to list up the definations. For example list of discription of different cities or list of discription of different places etc. now lets look how to create this list using HTML.

Creating HTML Discription list:

To Create a Discription list we have <dl> tag. And to add the discription term in the lists there is <dt> tag. after this add <dd> tag to add the discription for that term. Add anather <dt> to add anather discription term and <dd> to add its discription and carry on the same process as you want.
